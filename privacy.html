Privacy Policy

Ringkasan Singkat: Kami tidak mengumpulkan, menyimpan, atau mengirim data pribadi Anda. Semua proses JSON beautifier berjalan langsung di browser Anda (client-side only).

1. Informasi yang Kami Kumpulkan

VIN NESIA JSON Beautifier berkomitmen untuk melindungi privasi pengguna. Kami TIDAK mengumpulkan, menyimpan, atau memproses data pribadi apa pun dari pengguna, termasuk:

  • File JSON yang Anda upload atau input
  • Konten teks yang Anda masukkan
  • Informasi pribadi seperti nama, email, atau alamat
  • Data sensitif dalam file JSON Anda

2. Pemrosesan Data Client-Side

Semua fungsionalitas JSON Beautifier berjalan sepenuhnya di browser Anda (client-side processing):

  • File JSON diproses langsung di perangkat Anda
  • Tidak ada data yang dikirim ke server kami
  • Tidak ada data yang disimpan di server external
  • Semua operasi beautify, validasi, dan download berjalan offline di browser

3. Google Analytics

Kami menggunakan Google Analytics (GA4) untuk memahami penggunaan website dan meningkatkan layanan. Data yang dikumpulkan bersifat anonim dan meliputi:

  • Jumlah pengunjung dan halaman yang dilihat
  • Informasi browser dan perangkat (tanpa identifikasi personal)
  • Negara dan kota asal pengunjung (berdasarkan IP address)
  • Interaksi dengan fitur-fitur website (klik tombol, upload file)

Data ini digunakan untuk analisis statistik dan tidak dapat digunakan untuk mengidentifikasi individu tertentu.

4. Cookies dan Local Storage

Website ini menggunakan cookies minimal yang diperlukan untuk:

  • Google Analytics untuk pelacakan penggunaan website
  • Preferensi browser untuk pengalaman pengguna yang lebih baik

Kami TIDAK menggunakan cookies untuk menyimpan data JSON atau informasi sensitif Anda.

5. Keamanan Data

Karena semua pemrosesan dilakukan di browser Anda:

  • Data JSON Anda tidak pernah meninggalkan perangkat Anda
  • Tidak ada risiko data breach dari server kami
  • Anda memiliki kontrol penuh atas data yang diproses
  • Koneksi website menggunakan HTTPS untuk keamanan

6. Perlindungan Anak

Layanan ini tidak dirancang khusus untuk anak-anak di bawah 13 tahun. Kami tidak secara sengaja mengumpulkan informasi pribadi dari anak-anak. Jika Anda adalah orangtua dan mengetahui bahwa anak Anda memberikan informasi pribadi kepada kami, silakan hubungi kami.

7. Link ke Website Lain

Website ini mungkin berisi link ke website lain. Kami tidak bertanggung jawab atas praktik privasi website lain. Kami mendorong Anda untuk membaca kebijakan privasi setiap website yang Anda kunjungi.

8. Hak Pengguna

Karena kami tidak mengumpulkan data pribadi, tidak ada data pribadi yang perlu dihapus atau diperbaiki. Namun, Anda memiliki hak untuk:

  • Menonaktifkan cookies di browser Anda
  • Menggunakan mode incognito/private browsing
  • Menghubungi kami jika ada pertanyaan privasi

9. Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan diposting di halaman ini dengan tanggal revisi yang baru. Penggunaan berkelanjutan atas layanan kami setelah perubahan berarti Anda menerima kebijakan privasi yang diperbarui.

10. Google AdSense Compliance

Website ini mematuhi kebijakan Google AdSense:

  • Tidak mengumpulkan informasi yang dapat diidentifikasi secara pribadi
  • Menggunakan Google Analytics sesuai dengan Terms of Service Google
  • Memberikan transparansi penuh tentang penggunaan data
  • Mematuhi GDPR dan regulasi privasi internasional
