VIN NESIA JSON Beautifier

Ringkasan Singkat: Kami tidak mengumpulkan, menyimpan, atau mengirim data pribadi Anda. Semua proses JSON beautifying dilakukan sepenuhnya di browser Anda (client-side). File yang Anda upload tidak pernah meninggalkan perangkat Anda.

1. Informasi yang Kami Kumpulkan

JSON Beautifier Online berkomitmen untuk melindungi privasi pengguna. Kami TIDAK mengumpulkan, menyimpan, atau memproses data pribadi apapun, termasuk:

  • Nama, email, atau informasi identitas pribadi
  • File JSON atau konten yang Anda upload/input
  • Data hasil beautify/format JSON
  • Informasi login atau akun (karena tidak ada sistem akun)

2. Bagaimana Aplikasi Bekerja

Client-Side Processing

Semua pemrosesan JSON dilakukan sepenuhnya di browser Anda (client-side):

  • File yang Anda upload dibaca langsung oleh JavaScript di browser
  • Proses beautify/format menggunakan fungsi JSON.stringify() browser
  • Tidak ada data yang dikirim ke server atau pihak ketiga
  • File dan hasil processing hanya ada di memori browser Anda

Tidak Ada Penyimpanan Data

Aplikasi ini tidak menggunakan:

  • Database atau server penyimpanan
  • localStorage atau sessionStorage browser
  • Cookies untuk menyimpan data pengguna
  • Cache file atau konten pengguna

3. Google Analytics

Kami menggunakan Google Analytics (GA4) dengan ID tracking G-3PYZXZFNER untuk menganalisis penggunaan website secara anonim:

Data yang Dikumpulkan GA4:

  • Informasi teknis browser (jenis browser, sistem operasi)
  • Lokasi geografis umum (negara/kota, bukan alamat spesifik)
  • Halaman yang dikunjungi dan durasi kunjungan
  • Referrer/sumber traffic
  • Interaksi dengan elemen website (klik tombol, dll.)

Data yang TIDAK Dikumpulkan GA4:

  • Konten file JSON yang Anda upload/input
  • Informasi pribadi yang dapat mengidentifikasi Anda
  • Data sensitif atau rahasia bisnis

Anda dapat menonaktifkan Google Analytics dengan menggunakan ekstensi browser seperti "Google Analytics Opt-out" atau mengatur browser untuk memblokir tracking.

4. Cookies dan Teknologi Pelacakan

Website ini menggunakan cookies minimal yang diperlukan untuk:

  • Google Analytics (untuk statistik penggunaan anonim)
  • Preferensi teknis browser (tidak menyimpan data pribadi)

Kami tidak menggunakan cookies untuk:

  • Menyimpan data pribadi atau file pengguna
  • Tracking lintas website
  • Advertising atau marketing
  • Profiling pengguna

5. Keamanan Data

Keamanan Maksimal: Karena semua proses dilakukan client-side, data Anda tidak pernah meninggalkan perangkat Anda. Ini adalah metode paling aman untuk memproses file JSON sensitif.

Praktik Keamanan:

  • Koneksi HTTPS untuk mengenkripsi komunikasi
  • Tidak ada penyimpanan data di server
  • Tidak ada transmisi file ke server external
  • Code open-source yang dapat diaudit

6. Hak Pengguna

Karena kami tidak mengumpulkan atau menyimpan data pribadi, Anda secara otomatis memiliki kontrol penuh atas data Anda:

  • Hak Akses: Tidak ada data pribadi yang disimpan untuk diakses
  • Hak Hapus: Refresh browser akan menghapus semua data lokal
  • Hak Portabilitas: Data tetap di perangkat Anda
  • Hak Pembatasan: Anda dapat berhenti menggunakan layanan kapan saja
