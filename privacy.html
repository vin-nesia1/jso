Kebijakan Privasi JSON Beautifier Online

1. Informasi yang Kami Kumpulkan

JSON Beautifier Online berkomitmen untuk melindungi privasi pengguna. Kami TIDAK mengumpulkan, menyimpan, atau mengirim data pribadi atau file JSON Anda ke server manapun.

Data yang TIDAK Kami Kumpulkan:

  • File JSON yang Anda upload atau input
  • Konten JSON yang diproses
  • Informasi pribadi atau identitas
  • Data login atau akun
  • Alamat IP atau lokasi spesifik

Data Analytics (Anonim):

Kami menggunakan Google Analytics (GA4) untuk memahami penggunaan website secara umum:

  • Jumlah pengunjung dan halaman yang dilihat
  • Negara asal pengunjung (bukan alamat spesifik)
  • Jenis perangkat dan browser yang digunakan
  • Durasi kunjungan dan interaksi dengan fitur
  • Data demografis umum dan minat (jika tersedia)

2. Bagaimana Kami Menggunakan Informasi

Semua pemrosesan JSON dilakukan secara client-side (di browser Anda), yang berarti:

  • File JSON tidak pernah meninggalkan perangkat Anda
  • Tidak ada transmisi data ke server kami
  • Proses beautify dilakukan sepenuhnya offline
  • Data hanya tersimpan sementara di memori browser

Data analytics digunakan untuk:

  • Memahami pola penggunaan website
  • Meningkatkan kualitas layanan
  • Mengoptimalkan performa website
  • Mengidentifikasi dan memperbaiki bug

3. Penyimpanan Data

Kami TIDAK menyimpan data apapun di server. Semua data yang Anda masukkan:

  • Hanya tersimpan sementara di browser Anda
  • Akan hilang ketika Anda menutup atau refresh halaman
  • Tidak disimpan dalam database atau server manapun
  • Tidak dibagikan kepada pihak ketiga

4. Cookies dan Teknologi Pelacakan

Website ini menggunakan cookies untuk:

  • Google Analytics: Mengumpulkan data penggunaan anonim
  • Preferensi Browser: Menyimpan pengaturan lokal (jika ada)

Anda dapat menonaktifkan cookies melalui pengaturan browser, namun beberapa fitur mungkin tidak berfungsi optimal.

5. Keamanan Data

Keamanan data terjamin karena:

  • Semua proses dilakukan di browser Anda (client-side)
  • Tidak ada transmisi data sensitif
  • Website menggunakan HTTPS untuk enkripsi
  • Tidak ada penyimpanan data di server

6. Hak Pengguna

Sebagai pengguna, Anda memiliki hak untuk:

  • Menggunakan layanan tanpa memberikan data pribadi
  • Menonaktifkan Google Analytics melalui browser
  • Menghapus cookies dan data browser kapan saja
  • Berhenti menggunakan layanan tanpa konsekuensi

7. Layanan Pihak Ketiga

Google Analytics:

Kami menggunakan Google Analytics untuk analisis website. Google dapat mengumpulkan data sesuai dengan kebijakan privasi Google.

Google Fonts:

Website ini memuat font dari Google Fonts yang dapat mengirim permintaan ke server Google.

8. Perlindungan Anak

Layanan ini tidak secara sengaja mengumpulkan informasi dari anak-anak di bawah 13 tahun. Jika Anda adalah orang tua dan mengetahui bahwa anak Anda telah memberikan informasi pribadi, silakan hubungi kami.

9. Tautan Eksternal

Website ini mungkin berisi tautan ke website lain. Kami tidak bertanggung jawab atas praktik privasi website tersebut. Kami menyarankan Anda membaca kebijakan privasi setiap website yang dikunjungi.

10. Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan diumumkan di halaman ini dengan tanggal pembaruan yang baru.

11. Yurisdiksi dan Hukum yang Berlaku

Kebijakan privasi ini diatur oleh hukum Republik Indonesia. Setiap sengketa akan diselesaikan melalui pengadilan yang berwenang di Indonesia.
